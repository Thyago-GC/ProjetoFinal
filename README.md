## Projeto

 - API restful para cadastro de produtos

##  Configurações do projeto
  - Java 11
  - Banco H2
  - Spring-Boot (2.5.5)
  - Swagger
    - Springfox (2.9.2)
    - http://localhost:9999/swagger-ui.html
  - Port : 9999

      - Model
  - Documentação
    - Swagger
  - Tratamento de Exceções
